【CTF入门】常用工具的环境搭建（Python篇）

2022/5/16

1. Python篇

​ 对于ctf中常用的解密方法有很多，而脚本编写则绕不开python这一工具。但对于大多新手python环境的构建是一项令人头疼的事情，要不断的去网上找资源，去比对网上的电脑和自己电脑的区别，而本篇则是笔者在安装python时的经验。

​ 另：现阶段已经大部分在使用python 3了，但做一些老题目时可能会遇到从网上down下来的代码放入python 3中跑不起来的情况（毕竟python 2的版本也才2020年结束更新）所以在这里python 2的安装也会介绍。

Python 3

对于python的下载直接去网上搜索python 官网进行下载。

由于官网的页面形式改变所以只能靠读者自己下载了。在这里我只对当前的页面进行讨论。

选择download的releases 进入下一个页面，滑动滚轮到

去选择自己想要的版本。这里推荐python 3 的最新版本就可以了。


在这里选择合适的版本进行下载。

（在这里下载可能会有点慢，读者可以考虑科学上网，当然没有可以科学上网的方式笔者这里推荐一个软件 steam++对于打游戏的应该挺熟悉的但它还可加载国外一些网页的读取速度，比如/github之类的。下载方式呢直接去B站搜）。

下载好后安装到这个页面


在这里 记得选中Add Python 3.8 to PATH 这个选项，他会自行环境变量安装，不用手动安装（感动，能够自行安装环境变量的软件是个好软件！！）。

同时选中自定义安装也就是Customize installation（作为学习计算机的那肯定逼格要高点啊）。


在这里全选就可以了，尤其是pip这个选项，他作为一个工具包在日后的学习还是很重要的。

NEXT 继续


前面勾选中第四个一定勾选（会添加到环境变量）

这里需要更改下安装地址Customize install location

推荐安装在C盘这样后续会方便很多，当然C盘爆红了的话安装在其他盘也没问题。但在这里需要注意的是将文件安装在表层（这样会避免日后很多麻烦）比如笔者这个应该跟改为C:\python 就可以了。同时命名文件的时候应该避免出现中文，特殊字符，空格之类的。

继续安装。


这里会提示安装成功 Setup was successful。

同时下面会出现一行 Disable path length limit 提示字样。

点击这个地方（禁用文件深度最大限制，之后可能会对第三方库的时候会报错，那时候就要搞注册表之类的了很麻烦）

至此Python的安装就完成了。

当想要验证是否安装完成那就要去打开命令提示符（在windows的搜索中搜索cmd）

输入指令 python –version

C:\Users\用户名>python --version
Python 3.10.4

会出现python版本。

同时对于上文中的pip验证也是利用 pip –version 

C:\Users\用户名>pip --version
pip 22.0.4 from C:\文件地址 (python 3.10)

Python 2

对于python 2 来说安装步骤相较于python 3麻烦了很多，而且可能还会遇到环境变量被自己误删了的情况（别问我怎么知道的）所以我尽量简单的陈述安装过程。

安装同样也要去官网安装

选择download的releases 进入下一个页面，滑动滚轮到

在这里的话选择 python 2.7.18（因为这是python 2 的最后一个版本）


选择自己合适的版本（这里因为我的电脑是win 10 64位）所以选择的是这个。

打开安装包

这里的话两个随便选问题都不大。Next

这里可以更改安装包位置，同时建议安装再C盘，以及将其安装再文件表层，例如我这样的安装方式。Next

Next。

然后就安装完成，但在这地方还需要去添加Python的环境变量。

在windows的搜索中搜索编辑系统环境变量。打开后点击环境变量

找到PATH

双击进入PATH的编辑环境

去新建输入python的安装地址（当然也可以浏览到python的文件地址，笔者这里就只是放在了E盘中的python中所以直接浏览到这个文件加选中就行）

另选新建输入python安装文件的子级文件Scripts可以了（同样可以选择新建的时候浏览）。

当然可能会出现这样的提示，这里和上文的没太大区别，在这里只需要去手动输入自己安装的python文件地址，同时再每一个环境变量前后都要加 ; 就可以了。但要注意这里是添加而不是替换。

当然问题来了，如果不小心删除了系统原来的环境地址怎么办

记得千万别重启然后参考这个老哥的做法。

我这里的方法是重新添加PATH这个变量 并输入

%SystemRoot%\system32
%SystemRoot%

这两个变量。然后再尝试进行python地址的输入。

python 2 的验证如同 python 3 的验证一样

输入指令 python –version

C:\Users\用户名>python --version
Python 3.10.4

会出现python版本。

同时对于上文中的pip验证也是利用 pip –version 

C:\Users\用户名>pip --version
pip 22.0.4 from C:\文件地址 (python 3.10)

至此 python 2 的安装就完成了。
